# Fakebook API

## API

| **Method** | **API Endpoint**  | **Access** | **Description**      |
| ---------- | ----------------- | ---------- | -------------------- |
| /POST      | /api/users/signup | Public     | Create a new accoutn |
| /POST      | /api/users/login  | Public     | Login to app         |
| /POST      | /api/users/logout | Protected  | Logout of app        |
|            |                   |            |                      |
