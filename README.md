# Fakebook API

## API

| **Method** | **API Endpoint** | **Access** | **Description**      |
| ---------- | ---------------- | ---------- | -------------------- |
| /POST      | /api/users/login | Public     | Login to app         |
| /POST      | /api/user/logout | Protected  | Logout of app        |
| /POST      | /api/user/signup | Public     | Create a new accoutn |
|            |                  |            |                      |
